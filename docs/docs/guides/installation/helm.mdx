
1. [Install Helm](https://helm.sh/docs/intro/install/)

2. Install a ZITADEL Helm release
```bash
helm repo add zitadel https://zitadel-charts.storage.googleapis.com/zitadel
helm install my-zitadel zitadel/zitadel
```

Read more about the configuration options on [GitHub](https://github.com/zitadel/zitadel-charts) or on [ArtifactHub](https://artifacthub.io/packages/helm/zitadel/zitadel)
