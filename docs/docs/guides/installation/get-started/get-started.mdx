# Run ZITADEL

Choose your platform and run ZITADEL with the most minimal configuration possible.
For an easy self-hosted production setup, we recommend running ZITADEL on Kubernetes, using our official Helm chart.
By default, it runs a highly available ZITADEL instance along with a secure and highly available CockroachDB instance.
